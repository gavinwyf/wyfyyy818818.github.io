  8.    /* IE下的样式 */ 
p span{
display: block;
width:200px;/*对宽度的定义,根据情况修改*/
overflow: hidden;
white-space: nowrap;
text-overflow: ellipsis;
}

    text-overflow属性仅是注解，当文本溢出时是否显示省略标记。并不具备其它的样式属性定义。我们想要实现溢出时产生省略号的效果。还必须定义：强制文本在一行内显示（white-space:nowrap）及溢出内容为隐藏（overflow:hidden）。只有这样才能实现溢出文本显示省略号的效果。

    /* FF 下的样式 */ 
p{clear:both;}
p span{
float:left;
max-width:175px; 将省略号空间留出 /*IE不能解释该属性，而FF可以*/
}
p:after{
content:"...";  
}

    关于p span 的宽度用了”max-width”这个属性，IE不能解释该属性，而FF可以,这样就避开了IE对SPAN宽度的重新应用。

去哪网面试：(9月17号 班夫酒店)

数据结构：

1.栈中取最大数和索引 时间复杂度为 O(1) push pop 只能直接进行比较 可设置个哨兵作为参照

2.二叉树： 哈夫曼编码 哈夫曼树 最优二叉树(带权路径长度WPL最小的二叉树)
每次找权值最小的两个作为合并节点，取权值和为一个新父节点，加入表中，再进行上述操作。。。
直到构造成哈夫曼树，左取0，右取1，进行编码。

2.对象 

var obj = {};  
var obj = new Object; 
上面两种创建对象完全一样
还有一种： var obj = create(parent);  parent为obj.prototype

function point(){...}
var p = new point();
p.__proto__ == point.prototype  //true   隐藏属性__proto__为对象构造器的原型

自定义的对象:
先有的构造器，才有的对象，才能给对象添加原型 ??我个人理解
var obj = {};
obj.prototype  //undefined
obj.constructor  // function Object(){ [native code] }
obj.__proto__   // Object{}  
obj.constructor.prototype  //Object{}
obj.constructor.prototype == obj.__proto__  //true
