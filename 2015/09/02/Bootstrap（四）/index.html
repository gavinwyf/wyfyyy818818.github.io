Bootstrap知识点（四）

四、Bootstrap提供的jQuery插件

这些插件基本上都可以有两种使用方法：

(1)调用js函数：  $(...).yyy( );
(2)声明扩展属性：    <button  data-xxx="yyy">

1. 下拉列表

1)data-toggle="dropdown"
2)$(....).dropdown( )

2. 警告框

1)data-dismiss="alert"

3. 工具提示框

使用工具提示框必需两步：

1)<xx title="工具提示的内容" data-toggle="tooltip">
2)$('[data-toggle=tooltip]').tooltip( )

4. 弹出框

使用弹出框必需两步：

1)<xx title="标题" data-content="内容" data-toggle="popover">
2)$('[data-toggle=popover]').popover( )

5. 模态对话框——小重点&小难点

模态：一个父窗口中打开了一个子窗口，若子窗口不关闭，则父窗口无法获得焦点，且父子窗口间可以传递数据——这样的子窗口称为模态（modal）窗口。

模态框必需满足如下的层级结构：

<div class="modal fade">    <!--全屏半透明遮罩层-->
      <div class="modal-dialog">    <!--定位、宽高设置的容器-->
              <div class="modal-content">    <!--对话框窗口的边框、背景、阴影、倒角-->

                  <div class="modal-header">
                      <button class="close" data-dismiss="modal">&times;</button>
                      <h3 class="modal-title">Modal title</h3>
                  </div>

                  <div class="modal-body">对话框的主体</div>

                  <div class="modal-footer">对话框的尾部</div>

              </div>
      </div>
</div>

6. 标签页

必需的结构：

<ul class="nav  nav-tabs">
        <li class="active">
        <a data-toggle="tab" href="#id1"></a>
        </li>
        <li><a ></a></li>
        <li><a ></a></li>
</ul>
<div class="tab-content">
        <div id="id1" class="tab-pane active"></div>
        <div></div>
        <div></div>
</div>

7. Bootstrap提供的JS插件——折叠效果( Collapse )

触发元素：
<a data-toggle="collapse" href="#id2">触发元素</a>
<button data-toggle="collapse" data-target="#id2">触发元素</button>

目标元素：
<div id="id2" class="collapse in">目标元素</div>
  ------------------------------------------------------------

应用：

(1)使用Collapse插件实现Accordion(手风琴)组件

(2)使用Collapse插件实现可以折叠收起的响应式导航栏——重点！

8. Bootstrap提供的JS插件——轮播广告(Carousel)

最基本的轮播广告：

<div id="id-XX" class="carousel"  data-ride="carousel">

    <ol class="carousel-indicators">   滚动小圆圈
        <li data-target="#id-XX" data-slide-to="0" class="active"></li>
        <li data-target="#id-XX" data-slide-to="1"></li>
        <li data-target="#id-XX" data-slide-to="2"></li>
     </ol>

    <div class="carousel-inner">   内容

        <div class="item active">
               <img src="">  
               <div class="carousel-caption">  图片描述信息
                ...
               </div>
        </div>

        <div class="item">
               <img src="">  
               <div class="carousel-caption">  图片描述信息
                ...
               </div>
        </div>
    </div>

  前翻按钮
  <a class="left carousel-control" href="#id-XX" data-slide="prev">
    <span class="glyphicon glyphicon-chevron-left"></span>
  </a>

  后翻按钮
  <a class="right carousel-control" href="#id-XX" data-slide="next">
    <span class="glyphicon glyphicon-chevron-right"></span>
  </a>

</div>

参数： interval pause wrap keyboard

设置轮播时间 data-interval=”2000” (默认为5000)

9. 附加导航(Affix)

<div data-spy="affix" data-offset-top="60" data-offset-bottom="200">
  ...
</div>

JS: 

$('#myAffix').affix({
  offset: {
    top: 100,
    bottom: function () {
      return (this.bottom = $('.footer').outerHeight(true))
    }
  }
})
