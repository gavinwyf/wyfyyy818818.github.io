HTML DOM整理：

HTML标签对象化：将网页中的每个元素都看作一个对象

标准DOM操作适合： 操作节点，如创建、删除、查找等
HTMLDOM操作适合： 操作属性，如读取或修改属性的值 

   属性：a.getAttribute('href') <=> a.href
样式属性： a.style.color=''; a.className='';

常用HTML DOM对象： 

Document -Image
         -Table -TableCell
            -TableRow
         -Form -Input
           -Select -Option
       -Textarea
     -Anchor

1. Image对象：

创建:  HTMLDOM: var newNode = new Image(); 
      标准DOM: var newNode = document.createElement("img"); 

常用属性： src , height , width

2. Table对象：

增加行： var newRow=table.insertRow(i); //返回TableRow对象
删除行： table.deleteRow(i);    //注意: 删除行必须先获得行对象的下标     
所有行的数组：table.rows;
最后一行：table.rows[table.rows.length-1];

TableRow对象:

      增加格： var newTd=row.insertCell(i);
      删除格： row.deleteCell(i);
      行中所有格的数组：row.cells; //返回TableCell对象
      最后一格： row.cells[row.cells.length-1];

行下标：  row.rowIndex;------专用于删除行！       TableRow对象
列下标：  cell.cellIndex:当前格在行中的位置下标     TableCell对象

TableCell对象： 代表一个HTML单元格

innerHTML cellIndex colSpan rowSpan

3. Form对象：

查找： var form=document.forms[下标/"name"/"id"];
找任意元素： var elem=form.elements[i];
找name属性元素： var elem/elems=form.name属性值
                 form["name属性值"]
常用属性： action,method,enctype,length,elements[]

API: 设置元素焦点： elem.focus();
     手动提交表单： form.submit();
 重置： form.reset()

4. Input对象：

                       HTML DOM对象
<input type="text">      Text        name/value  blur()/focus()/select()
<input type="password">  Password    name/value  blur()/focus()/select()
<input type="checkbox">  Checkbox    name/value/checked  blur()/focus()/click()
<input type="radio">     Radio       name/value/checked  blur()/focus()/click()
<input type="submit">    Submit      name/value/checked  blur()/focus()/click()
<textarea>               Textarea    name/value  blur()/focus()/select()

4. select对象：

(1)增加Option:2步： 
     var option=new Option(innerHTML,value);
     select.add(option);    //传统: document.createElement("option"); option.innerHTML=..;
     简写： select.add(new Option(innerHTML,value));

(2)删除Option: select.remove(index);

(3)数组：  options: 封装了所有option对象
          固定套路：删除最后一个选项：
      select.remove(select.options.length-1);

(4)特殊属性： select.selectedIndex  获得列表框当前选中项的下标
             固定套路：获得当前选中项的option对象
         select.options[select.selectedIndex];

(5)事件：onchange()

Option对象：

创建对象：var o = new Option(text,value);
常用属性：index/text/value/selected

5. cookie: 保存key/value对儿的.txt文件

由服务器生成并存储在客户端文件系统(.txt格式)中的键值对
当浏览器再次请求该站点上的页面时，就会自动把保存的Cookie发回服务器

作用：Cookie使浏览器可在访问同一个站点的不同请求间传递数据，让服务器识别不同的客户端

有效期(生命周期)：

(1) expires值，定义生存周期，周期内Cookie有效，超出周期Cookie会被删除。

(2) 有些页面设为0或负值，关闭浏览器时，马上清除Cookie，不会记录用户信息，更安全。

(3) 若没有expires属性，创建的Cookie保存在内存中，浏览器一关闭就销毁。

作用域：默认情况下，站点保存的Cookie只能自己访问。但可以设置Cookie的domain和path值，限定哪个域名下的哪个路径可以访问Cookie

(1)何时创建： 只要在客户端持久保存个人数据，就用cookie

(2)何时使用： 向网站发送请求时，浏览器会自动携带该网站的cookie一并发送。

(3)如何创建cookie:  document.cookie="key=value;expires="+date;

   eg: document.cookie='uname=Tom';
   document.cookie='msg='+encodeURIComponent('Hi,JS你好');
   document.cookie='uid=7788;expires='+new Date().toGMTString();
   创建并保存三个Cookie

(4)读取cookie： document.cookie
            每个键值对以分号空格分隔
            *不包含expires属性*               

(5)如何读取每个属性和属性值
        先用split("; ")分隔每个键值对
        找到指定key的元素
        获取key对应的value

   eg: var cookies=document.cookie;
       var arr=cookies.split(";");   
       for(var i=0;i<arr.length;i++){
             var cookie = arr[i];
          var cookiePair = cookie.split("=");
          console.log(cookiePair[0]+'='+cookiePair[1]); //输出键值对
       }
