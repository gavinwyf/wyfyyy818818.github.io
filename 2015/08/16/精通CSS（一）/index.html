精通CSS整理（一）：

这本书看完挺长时间了，但是记住的好像很少，一开始感觉这本书都是基础的东西，看一遍就可以，但是前一阵做响应式
布局的时候，发现有一些重要的东西在这本书中提到过。例如有关流式布局，弹性布局，弹性图片等，还有后面的一些
兼容性相关的东西，都挺重要的，我把这本书简要的整理一下。就当是复习一下CSS了。

第一章 基础知识

1.语义化命名约定 “它们是什么”来命名 微格式

2.有意义的标记 Web语义化

1）有意义的页面更容易处理
2）程序和其他设备也可以理解有意义的标签（也称语义标签）
3) 可以简便的将元素调整为你所需的样式。

3.ID和类名选择器的差异和适用场合

类： 应用于概念上相似的元素
ID:  应用于不同的唯一的元素

4.CSS、HTML版本

DTD（文档类型定义）   严格strict 过渡transitional

5.浏览器对不同版本的解析差异

浏览器模式：标准模式和混杂模式(quirks mode)

第二章 为样式找到应用目标

1.常用选择器

标签/元素 后代 ID 类 通用*

CSS3中： 

子选择器   >   
相邻同胞(兄弟)选择器   +  
通用兄弟选择器    ~   
 属性选择器    []   
 伪类/伪元素选择器

’

IE7和更高版本支持子选择器  IE7中有个小bug，若父元素和子元素之间有HTML注释，会出问题
IE6和更低版本，可用通用选择器模拟

伪类：（当然伪类还有很多，CSS3中有很多好用的伪类，等以后专门整理，伪类真的能做很多事）

动态伪类：(理论上应用于所有元素)
   :hover 鼠标悬停               IE6 只注意active和hover，完全忽略focus
   :active 激活的                IE7 在任何元素支持hover，但忽略active和focus
   :focus 选中时
链接伪类：(只用于锚元素)
    :link 未访问过
    :visited 访问过的

层叠：  简化CSS
继承:   应用样式的元素后代会继承样式的某些属性，eg. color font 
        减少代码中选择器的数量和复杂

第三章 可视化格式模型

1. 标准W3C盒模型： 内容、内边距、边框、外边距

使用全局reset把内边距和外边距显式的设置为零可能更安全

2. IE(混杂模式)盒模型： width = 内容 + 内边距 +边框

IE早期版本，包括IE6，在混杂模式中使用自己的非标准盒模型。

CSS3中，box-sizing 定义盒模型类型

3. 外边距叠加 垂直外边距取较大的

(前提是普通文档流中的块元素的垂直外边距) 

行内元素、浮动元素或绝对定位元素之间不会叠加

4. 定位 三种基本定位机制：普通流、浮动、绝对定位

relative: 相对于文档流中初始位置
absolute: 相对于最近已定位父元素               可用z-index 控制叠放次序
fixed：   相对于视口（固定定位是绝对定位的一种，IE6不支持，IE7部分支持有bug）
static:   默认
inherit:  从父元素继承

5. 浮动： 使元素脱离文档流

清理浮动： 实际上为之前浮动元素留出了垂直空间

   (1) clear:both;
   (2) overflow:hidden或auto  （有可能影响元素表现，如产生滚动条或截断内容）

 **常用这种方式**： 

   .clear:after{
         content:".";
           height:0;
         visibility:hidden;
         display:block;
         clear:both;
    }

第四章 背景图像效果

1.背景图片

background-position:left center; 左侧中间
background-position:0 50%; 垂直居中

2.圆角框

(1)定宽圆角：  顶+底

(2)灵活圆角：  四个角  滑动门技术(需定义两个额外的无语义元素)

(3)山顶角：  多个背景图像，拼成圆角
            border-radius(-moz- -webkit) 
            border-image (九分法缩放)

3.投影

(1)简单的CSS投影(负外边距)

(2)Clagnut投影(相对定位)   left:-5px;top:-5px;

(3)CSS3: Box-shadow: 3px 3px 6px #666

4.不透明度

(1)三种方式： opacity  RGBa   PNG透明度(IE6不直接支持PNG透明度，IE7/IE8支持)

(2)CSS视差效果: ( background-color:transparent; )

5 图像替换： 

FIR ( display:none )
Phark (  text-indent: -5000px; )
sFIR (可伸缩Inman Flash替换)   过时已久了，就不想多说了

第五章 对链接应用样式

1.链接选择器（层叠顺序）

link , visited 将覆盖之前的 hover , active

最好按照次序使用： a:link,a:visited,a:hover,a:focus,a:active

2.链接修饰： 主要使用background属性

3.链接目标设置样式： :target伪类(唯一可以不用JS使其他元素产生样式变化的方法)

4.突显不同链接(属性选择器)： a[att^=val] 标识外部链接

eg:   a[href^="http:"]  "mailto:" "aim:"
      a[href^=".pdf"]   a[href$=".rss"]RSS提要(feed)  ".doc"  ".rdf"

IE6不支持属性选择器。

5.链接应只用于get请求，决不要用于post

6.图像翻转： background-image background-position

CSS精灵： 图标，按钮，图形，集中在一个地方  使用background-position

CSS3中： text-shadow,box-shadow,border-radius

倒影 -webkit-box-reflect: below 2px -wekit-gradient(linear,left top,left bottom,
