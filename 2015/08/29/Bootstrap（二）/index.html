Bootstrap知识点（二）

二、全局CSS样式————几百个class

1.容器

(1)container

.container {
    width:   1170px  或者 970px 或者 750px; 
    padding-left:  15px;
    padding-right:  15px; 
    margin-left: auto;
    margin-right: auto;
    box-sizing: border-box;
}

.container:before {        /*防止子元素的margin-top超出父元素边界*/
    content: " ";
    display: table;
}

.container:after {        /*防止子元素浮动对后续元素造成影响*/
    content: " ";
    display: table;
    clear: both;
}

(2)container-fluid

.container-fluid {    流式容器
    width:  auto; 
    padding-left:  15px;
    padding-right:  15px; 
    margin-left: auto;
    margin-right: auto;
    box-sizing: border-box;
    }
    .container-fluid:before {
    ....
    }
    .container-fluid:after {
    ....
}

4.表格相关样式——小重点

.table    设定表格的宽度和空白
.table-bordered   添加最外层的边框
.table-responsive    响应式表格，添加在<table>的父元素<div>上！
.table-striped    隔行变色
.table-hover  悬停变色
.table-condensed  紧凑些的表格
还可以使用下述class用于tr和td，添加背景色：
.active
.danger
.success
.info
.warning

5.栅格布局系统——重点

页面布局的方式：

(1) TABLE布局

(2) DIV+CSS布局

(3) Bootstrap提供的布局：(1)+(2)的优势！

Bootstrap栅格布局系统：

(1) 有行的概念：  .row        有列的概念：   .col

(2) 行必须处于.container/.container-fluid中，一行平均分为12列

(3) 行中只能放置列，内容只能放在列中

(4) 列中还可以继续嵌套行，嵌套的行又可以分为12列

相关class：

 .row        新换一行
 .col-lg-1  ~  .col-lg-12      在大屏下(>=1200)有效
 .col-md-1  ~  .col-md-12  在中屏/大屏下(>=992)有效
 .col-sm-1  ~  .col-sm-12   在小屏/中屏/大屏下(>=768)有效
 .col-xs-1  ~  .col-xs-12  在超小/小屏/中屏/大屏下有效
 ------------------------------
 .col-lg-offset-0  ~  .col-lg-offset-12     偏移n个单元格
 .col-md-offset-1  ~  .col-md-offset-12
 .col-sm-offset-1  ~  .col-sm-offset-12
 .col-xs-offset-1  ~  .col-xs-offset-12  
--------------------------------
 .hidden-lg        仅在大屏下隐藏     
 .hidden-md        仅在中屏下隐藏
 .hidden-sm        仅在小屏下隐藏
 .hidden-xs        仅在超小屏下隐藏

6.表单相关样式——小重点

容器元素可以使用的class：

.form-group    用于包裹label和输入框
.checkbox    用于包裹 label和复选框
.radio        用于包裹lable和单选框
---------------------------
.form-control    指定输入框的样式(input)
---------------------------

三种表单：

普通表单：      <form>
行内表单：    <form class="form-inline">
水平表单：    <form class="form-horizontal">

注意：水平表单需要对label和input做单元格宽度分配！

7、 辅助类

文本颜色：

<p class="text-muted">...</p>
<p class="text-primary">...</p>
<p class="text-success">...</p>
<p class="text-info">...</p>
<p class="text-warning">...</p>
<p class="text-danger">...</p>

背景颜色：

<p class="bg-primary">...</p>
<p class="bg-success">...</p>
<p class="bg-info">...</p>
<p class="bg-warning">...</p>
<p class="bg-danger">...</p>

关闭按钮：

<button type="button" class="close"><span aria-hidden="true">&times;</span></button>

三角符号：

<span class="caret"></span>

快速浮动：

<div class="pull-left">...</div>
<div class="pull-right">...</div>

 .pull-left {
   float: left !important;
 }
 .pull-right {
   float: right !important;
 }

内容居中：

  <div class="center-block">...</div>

.center-block {
  display: block;
  margin-left: auto;
  margin-right: auto;
}

显示与隐藏：

 <div class="show">...</div>
 <div class="hidden">...</div>


.show {
  display: block !important;
}
.hidden {
  display: none !important;
  visibility: hidden !important;
}
.invisible {
  visibility: hidden;
}
