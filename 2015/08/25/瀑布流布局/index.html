瀑布流布局

一、瀑布流布局：

它的特点是视觉表现的参差不齐的多栏布局，随着页面滚动条向下滚动不断加载数据到当前尾部。

最早采用瀑布流布局的是国外的pinterest网站，现在国内也相当很多。例如蘑菇街、美丽说。

二、实现方法：

JQuery 方法

CSS3的多栏布局

1.Javascript原生方法

采用 定位、浮动

图片外包裹两层盒子，大盒子处理盒子之间的布局关系，小盒子是图片的相框。

原理：

第一阶段：瀑布流的布局定位

1、计算列数=浏览器窗口宽度/图片宽度 图片定位使根据每一列数据库的高度计算接下来图片的位置

2、获取第一行图片的高度放到数组中，判断第二行第一张图片应出现的位置并做定位

3、第二行第一张图片要加在第一行最矮的图片下面
(第二行第一张的top值应该是第一行最矮的图片的高，left就是最矮图片的距最左边的距离）

4、获取最矮图片的索引值和高度值 getMinhIndex minH，设置第二行第一张位置

5、改变放第一行高度的数组，使最矮的图片高度改变为加上下面图片的高度和，使后面每一个图片都进行定位并改变高度。

第二阶段：瀑布流随滚动条动态添加

1、添加滚动条事件，监听滚动条

2、滚动条向下移动，页面向下移动，若最后一张图片加载一半时加载数据

3、获取最后一张图片加载一半时相对页面的高度，获取页面滚走高度和浏览器窗口可视区高度，判断是否加载图片

lastBoxH < scrollTop+height时，加载图片
//图片加载一半时页面高度
lastBoxH=oBoxs[oBoxs.length-1].offsetTop+Math.floor(oBoxs[oBoxs.length-1].offsetHeight/2);
//页面滚走的距离
scrollTop=document.body.scrollTop||document.documentElement.scrollTop;
//浏览器窗口可视区域的高度
height=document.body.clientHeight||document.documentElement.clientHeight;

4、自定义一个JSON对象，提供加载图片信息，动态添加图片到页面

for(var i=0;i<dataInt.data.length;i++){
          var oBox = document.createElement("div");
          oBox.className='box';
         oParent.appendChild(oBox);
         var oPic = document.createElement("div");
         oPic.className='pic';
         oBox.appendChild(oPic);
         var oImg = document.createElement('img');
         oImg.src="img/"+dataInt.data[i].src;
         oPic.appendChild(oImg);
}

2.JQuery方法

第一阶段：瀑布流的布局定位

原生JS方法中很多方法都可以省略掉，比如 获取对应类名的div方法，获取最矮图片索引方法。

具体代码如下：

$(window).on('load',function(){    waterfall();    })
function waterfall(){
  var $boxs=$('#main>div');
  var w=$boxs.eq(0).outerWidth();  //outerWidth()把padding、border都包含在内，width()则不会
  var cols=Math.floor($(window).width()/w);
  $('#main').width(w*cols).css('margin','0 auto');
  var hArr=[];
  $boxs.each(function(index,value){
      var h=$boxs.eq(index).outerHeight();
      if(index<cols){
          hArr[index]=h;
      }else{
          var minH=Math.min.apply(null,hArr);
          var minHIndex=$.inArray(minH,hArr);
          $(value).css({
            'position':'absolute',
            'top':minH+'px',
            'left':minHIndex*w+'px'
          })
          hArr[minHIndex]+=$boxs.eq(index).outerHeight();
      }
  })
}

第二阶段：瀑布流随滚动条动态添加

两大特点：链式操作，隐式迭代 (连续使用方法作用在同一个对象)

lastBoxDis = $lastBox.offset().top+Math.floor($lastBox.outerHeight()/2);
scrollTop = $(window).scrollTop();
documentH = $(window).height();

lastBoxDis < scrollTop+documentH时，动态添加图片

$.each(dataInt.data,function(key,value){       //dataInt是JS原生对象，不能直接使用value
      var oBox = $("<div>").addClass("box").appendTo($("#main"));
      var oPic = $("<div>").addClass("pic").appendTo($(oBox));
      $('<img>').attr('src',"img/"+$(value).attr('src')).appendTo($(oPic));
})

3. CSS3 方法

前两种都是用到了绝对定位实现的。

CSS3使用多栏布局方法实现，column….

三、瀑布流三种方法比较

JS:

1、需要计算 计算列数=浏览器窗口宽度/图片宽度 图片定位使根据每一列数据库的高度计算接下来图片的位置

2、图片排序是按照图片计算的位置横向排列，位置是计算出来的，比较规范

CSS:

1、不需要计算，浏览器自动计算，只需要设置列宽，性能高

2、列宽随着浏览器窗口大小进行改变，用户体验不好

3、图片排序按照垂直顺序排列，打乱图片显示顺序

4、图片加载还是依靠Javascript实现
