Bootstrap知识点（三）

三、 Bootstrap组件

1.Bootstrap组件——Glyphicon图标字体

Glyphicon Halflings 字体是第三方提供的，Bootstrap项目可以作为服务器端字体免费使用。

<span class="glyphicon glyphicon-XXX"></span>  可以出现在任何使用文字的地方

2.按钮组

水平排布的按钮组

<div class="btn-group">...</div>

两端对齐的按钮组

<div class="btn-group btn-group-justified">
<a >...</a>
</div>

竖直排布的按钮组

<div class="btn-group-vertical">...</div>

大号按钮组gl、小号按钮组sm、超小号按钮组xs….

3.下拉菜单

下拉菜单必需三级结构：

<div class="dropdown">

< button data-toggle2="dropdown3">
      触发元素
</button>

<ul class="dropdown-menu">
      下拉菜单(隐藏元素)
    < li class="divider"></li>    
    < li class="dropdown-header"></li>
</ul>
</div>

4.输入框组

 输入框前后加提示文本： 
 <div class="input-group">
        <span class="input-group-addon"></span>
        <input class="form-control">
 </div>

输入框前后加按钮：
 <div class="input-group">
        <input class="form-control">
        <span class="input-group-btn">    注意！！！
        <button>...</button>
        </span>
 </div>

5.导航

  标签页式导航：
<ul class="nav  nav-tabs  nav-justified">
    <li class="active"></li>
</ul>

  胶囊式导航：
<ul class="nav  nav-pills  nav-justified">
    <li class="active"></li>
</ul>

  竖直放置的导航
<ul class="nav  nav-pills  nav-stacked">
    <li class="active"></li>
</ul>

6.进度条

进度条必需2级结构

<div class="progress">
    < div class="progress-bar" style="width:40%">
           40%
    </div>
</div>

7.缩略图——常用于“商品列表”

<div class="thumbnail">
    <img >
    <div class="caption">
    </div>
</div>

注意：展示多个商品时，往往和栅格系统组合应用

8.警告框

<div class="alert  alert-danger...."></div>
  可以关闭的警告框（需要使用Bootstrap提供的jQ插件）

<div class="alert  alert-danger  alert-dismissible">
    <a class="close"  data-dismiss="alert">
    &times;
    </a>
</div>

9.导航条——难点&重点

一个网站的导航条中，可能包含：商标/LOGO、菜单、搜索框、登录注册等按钮/超链接——响应式网页的导航条在手机中显示时，应该只有一个LOGO、下拉菜单按钮

导航条按颜色可以分为两种：

navbar-default：灰底黑字
navbar-inverse：黑底灰色

导航条按位置可以分为两种：

随内容滚动的导航条
固定在顶部/底部的导航条

例如：

<div class="navbar navbar-default">
    <div class="container">

        <!--导航条头部：LOGO+下拉菜单按钮-->
        <div class="navbar-header">
        LOGO + BUTTON
        </div>

        <!--导航条主体部分：菜单+搜索框+链接...-->
        <ul class="navbar-nav">菜单</ul>
        <form class="navbar-form"></form>
        <a class="navbar-link"></a>
    </div>
</div>

10.路径导航（面包屑）

 <ol class="breadcrumb">
      <li><a href="#">首页</a></li>
      <li><a href="#">产品大全</a></li>
      <li><a href="#">大家电</a></li>
      <li><a href="#">冰箱</a></li>
      <li><a href="#">三门冰箱</a></li>
</ol>

11. 分页

<ol class="pagination">
  <li><a href="#">&laquo;</a></li>
  <li><a href="#">1</a></li>
  <li><a href="#">2</a></li>
  <li class="active"><a href="#">3</a></li>
  <li><a href="#">4</a></li>
  <li><a href="#">5</a></li>
  <li><a href="#">&raquo;</a></li>
</ol>

12. 标签

< span class=”label  label-default”>New</span>

13. 徽章—— 提示消息的数量

< span class="badge">123</span>

14. 巨幕（宽大的屏幕）、——可用于中上部的大型广告

< div class="jumbotron">
  < h1>Hello, world!</h1>
  < p>...</p>
  < p>< a class="btn btn-primary btn-lg" href="#">Learn</a></p>
</div>

15. 井(Well)——可用于“文章导读/摘要”

< div class="well"></div>
< div class="well-lg"></div>
< div class="well-sm"></div>
< div class="well-xs"></div>

16.媒体对象——可用于”评论”、”产品列表”

 <div class="media">

      <div class="media-left">
        <a href="#">
          <img class="media-object" src="..." alt="...">
        </a>
      </div>

      <div class="media-body">
        <h4 class="media-heading">Media heading</h4>
        ...
      </div>
</div>

17.面板——可用于面板组

带标题：

  <div class="panel panel-default">
  <div class="panel-heading">
    <h3 class="panel-title">Panel title</h3>
  </div>
  <div class="panel-body">
    Panel content
  </div>
</div>

带注脚：

<div class="panel panel-default">
  <div class="panel-body">
    Panel content
  </div>
  <div class="panel-footer">Panel footer</div>
</div>

不同颜色：

<div class="panel panel-primary">...</div>
<div class="panel panel-success">...</div>
<div class="panel panel-info">...</div>
<div class="panel panel-warning">...</div>
<div class="panel panel-danger">...</div>

带表格：

<div class="panel panel-default">

      <!-- Default panel contents -->
      <div class="panel-heading">Panel heading</div>
      <div class="panel-body">
        <p>...</p>
      </div>

      <!-- Table -->
      <table class="table">
        ...
      </table>
</div>

带列表组的面板：

<div class="panel panel-default">

      <!-- Default panel contents -->
      <div class="panel-heading">Panel heading</div>
      <div class="panel-body">
        <p>...</p>
      </div>

      <!-- List group -->
      <ul class="list-group">
        <li class="list-group-item">Cras justo odio</li>
        <li class="list-group-item">Dapibus ac facilisis in</li>
        <li class="list-group-item">Morbi leo risus</li>
        <li class="list-group-item">Porta ac consectetur ac</li>
        <li class="list-group-item">Vestibulum at eros</li>
      </ul>
</div>
