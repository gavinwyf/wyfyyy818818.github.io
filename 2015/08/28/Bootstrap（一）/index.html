Bootstrap知识点（一）

  1. Bootstrap是一个HTML/CSS/JS框架，简化移动设备优先的响应式Web开发。

  2. Bootstrap的内容涉及到如下三个方面：

    (1) HTML： 为已有的H5标签新添了很多自定义属性 data-xxx

    (2)CSS： CSS Reset + 几百个class -> bootstrap.css

    (3)JS： 基于jQuery提供了几十个插件

  3. 知识点划分为五个部分：

    (1) 起步：      bootlint.js   html5shiv.js    respond.js

(2) 全局CSS样式：    容器、按钮、图片、表格、表单、栅格、排版、辅助类

(3) 组件：   字体图标、按钮组、下拉菜单、输入框组、导航、导航条、路径导航(面包屑)、分页、标签、徽章、巨幕、缩略图、警告框、进度条、媒体对象、列表组、面板、Well  

(4) jQ插件：  下拉菜单、按钮、模态框、标签页、工具提示、弹出框、警告框、Collapse(折叠框)、Carousel(轮播图)、Affix(附加侧边菜单)

(5) 定制：  LESS

一、 Bootstrap起步

1. 使用Bootstrap时涉及到几个文件：

(1) bootstrap.css  核心CSS文件，其中进行了CSSReset + 几百个class

(2) bootstrap.js   核心JS文件，基于jQuery提供了几十个插件

(3) html5shiv.js  由第三方提供，让老IE可以支持HTML5新标签

(4) respond.js  由第三方提供，让老IE可以支持CSS3新特性（媒体查询）

2. 基本模版

声明   <!DOCTYPE html>
元数据  <media name=”viewport”....>
样式   <link  rel=”stylesheet” href=”... ”>
脚本    <script src=”... ”></script>
兼容性   IE渲染： <meta http-equiv=”X-UA-Compatible” content=”IE=edge ”>
         高速模式： <meta name=”renderer ” content=”webkit ”>
         兼容IE：
           <!--[if lt IE 9]>
             <script src="js/html5shiv.min.js"></script>   IE支持HTML5
             <script src="js/respond.min.js"></script>     IE支持CSS3（尤其是媒体查询）
              <![endif]-->

3. Bootstrap模版：

DOCTYPE声明   父路径   元数据   样式引用   脚本引用   兼容性注释   容器

4. 禁用响应式布局

(1) 移除CSS文档中(viewport)的标签： <meta>

(2) 通过 .container类设置一个固定width值

(3) 需要移除所有导航条的折叠和展开行为

(4) 增加 .col-xs-* 类 或 替换掉 .col-md-*和.col-lg-*

6、box-sizing属性，定义如何计算盒子的总宽度：

一个盒子的总宽： 

content-box:      margin + border + padding + width
padding-box:     margin + border + width
border-box:       margin + width

基础按钮：       .btn
五种基础色：  .btn-danger                 四种基本大小：        .btn-lg
            .btn-success                                .btn-sm
            .btn-warning                                .btn-xs
            .btn-info
            .btn-primary

8、 Bootlint

Bootstrap官方提供的一个js工具，可以用于检测Bootstrap class使用时是否正确。

使用方法：

<script>
(function () {
    var s = document.createElement("script");
    s.onload = function () {
      bootlint.showLintReportForCurrentDocument([]);
    };
    s.src = "js/bootlint.js";
    document.body.appendChild(s)
  })();
</script>
